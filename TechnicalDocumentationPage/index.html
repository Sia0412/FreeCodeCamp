Introduction

HTML is the standard markup language for Web page.

CSS stands for Cascading Style Sheets and it saves a lot of works.

  • You can create your own website.
  • It is easy to learn.
HTML basic

There are some basic things to know about HTML. Like-

  • HTML is the standard markup language.
  • HTML describes the structure of a Web page.
  • HTML consists of a series of elements.
    • HTML elements tell the browser how to display the content.
  • HTML elements label pieces of content such as "this is a heading",etc.
Detail operations of HTML

Some of the operations with examples are given below-

  • The HTML element is everything from the start tag to the end tag.

    • <tagname>Content goes here...</tagname>

  • HTML headings are titles or subtitles.

    • <h1>heading 1</h1>

  • A paragraph always starts on a new line.

    • <p1>paragraph 1</p1>

  • Links allow users to click their way from page to page.

    • <a href="www.hstu.com">HSTU</a>

  • HTML tables allow web developers to arrange data into rows and columns.

    • <table><tr></tr></table>

CSS basic

There are some basic things to know about CSS. Like-

  • CSS describes how HTML elements are to be displayed on screen, paper, or in other media.
  • It can control the layout of multiple web pages all at once.
  • External stylesheets are stored in CSS files.
Detail operations of CSS

Some of the operations with examples are given below-

  • A CSS rule consists of a selector and a declaration block.

    • h1 {color:blue;font-size:12px}

  • A CSS selector selects the HTML element(s) you want to style.

    • -->In the upper example,"h1" is the header type selector.

  • Colors are specified using predefined color names, or RGB, HEX,etc.

    • body {color:cyan;}

  • The CSS background properties are used to add background effects for elements.

    • li {background:rgba(0, 128, 0, 0.3);}

  • Margins are used to create space around elements.

    • p {margin:15px;}

  • Padding is used to create space around an element's content.

    • head {padding:15px;}

Conclusion

Finally we can say that-

  • HTML cannot be used in a CSS file.
  • HTML is used to build the structure of the web pages.
  • CSS can be used in an HTML file.
  • CSS is used to make web pages more presentable.
